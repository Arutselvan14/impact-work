CSS COMBINATORS

a combinators is something that explains the relationship between the selectors

there are four different combinators in css

1.Descendant selectors(space)
the descendant selectors matches all elements that are descendants of a specified element

2.child selectors(>)
the child selectors select all elements that are the children of a specified element

3.adjacent sibiling selector(+)
the + selectors select all elements that are the children of a specified element

4.general sibiling selectors(~)
general sibiling selectors(~) selects all elements that are next sibilings of a specified element
